Design Rules
Never introduce additional brand colors.
Never replace orange with blue or purple.
Never replace the dark sidebar.
Never create gradients unless explicitly requested.
Avoid excessive glassmorphism.
Avoid neon effects.
Avoid colorful interfaces.
Whitespace is preferred over decoration.
Typography is preferred over heavy borders.
Consistency is more important than visual novelty.
Every screen must look like part of the same VALO-REST platform.
The orange active navigation state is the visual signature of VALO-REST and must remain consistent across Platform Owner, Restaurant Admin, Waiter, Kitchen Display System (KDS), and Cashier interfaces.

Modern
Brand colors remain unchanged.
UI should prioritize speed over decoration.
Avoid unnecessary animations.
Prefer whitespace over borders.
Prefer typography over decoration.
Every screen should communicate the primary action within two seconds.
